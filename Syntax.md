- Initialization
	`var <identifier> = <expression>;`

- Binary operation
	`operand1 <op> operand2`

	- Operations supported: `=`, `+`, `-`, `*`, `/`, `==`, `!`, `!=`, `<`, `<=`, `>`, `>=`

- Conditional
	`if <boolean expression> {`
	`...`
	`}`
	`else if <boolean expression> {`
	`...`
	`} else {`
	`...`
	`}`

- Control Flow
	`return <expression>;`

- Function Declaration
	`func <function name> (<parameter list>) {`
	`...`
	`}`

- Function Call
	`<function name> (<argument list>)`
